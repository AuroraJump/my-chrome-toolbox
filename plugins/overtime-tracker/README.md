# 加班工时实时统计 v2.2.3 - Chrome 扩展

## 安装
1. `chrome://extensions/` 打开开发者模式
2. 加载已解压 → 选本目录
3. 打开考勤页

## 加班规则(可在插件图标点开调整)

### 工作日
- **排班模式**:
  - `弹性` (默认): 加班 = max(0, 实际工时 − 目标工时 − 门槛)
  - `固定`: 加班 = max(0, 实际下班 − 标准下班 − 门槛)(与实际上班时间无关)
- **固定模式**: 上班时间 + 下班时间 (默认 09:00 → 18:00)
- **弹性模式**: 每日目标工时 (默认 9h)
- **加班门槛**: 延后超过此值才算加班, 否则 0 (默认 2h)

### 周末 / 节假日
- 不分弹性/固定模式, 统一规则:
  - 工时 > `weekendThreshold` (默认 4h) 才算加班
  - 单日封顶 `weekendMax` (默认 8h)

### 交通补贴(按天计,不算时长)
- **工作日**: 实际下班 ≥ `subsidyStartTime` (默认 21:00,含整点) → **1 次/天**
- **周末/节假日**:
  - 工时 > 4h → **1 次**
  - 工时 > 9h → **2 次**

### 算法示例 (threshold=2h, baseHours=9h, workEndTime=18:00)

| 打卡 | 弹性模式 | 固定模式 |
|------|----------|----------|
| 07:48-21:01 | 4h13m OT | 3h01m OT |
| 07:52-18:10 | 0 (未达门槛) | 0 (未达门槛) |
| 09:00-19:30 | 1h30m OT | 1h30m OT |

### 特殊日
- 法定节假日(无打卡) / 调休 / 年假 / 病假 / 事假 / 婚假 / 丧假 / 产假 / 陪产假 / 工伤假 / 未打卡 → 加班 = 0
- 6/1 等"系统误标节假日"但有实际打卡的 → 按工作日计算

## 悬浮窗功能
- 📊 实时显示本月累计加班
- 📈 目标完成度进度条
- 🔥 超过 100% 红色高亮 + 脉动动画
- 📅 可展开每日明细
- 🚕 交通补贴次数(显示在主面板, 明细里每条 OT 末尾 `· 🚕×N`)
- ⏰ 该打卡了下班提醒(工作日有上班无下班时显示)
- 🖱️ 标题栏拖拽
- ➕➖ 折叠 / 展开
- 🏷️ 右上角 OT 红标(诊断标记, 点开显示版本号)

## 文件
- `manifest.json` - 扩展配置(版本号)
- `content.js` - 核心: 读日历 + 计算 + 渲染
- `content.css` - 悬浮窗样式
- `popup.html / popup.js / popup.css` - 规则配置弹窗
- `diagnose.js` - 诊断脚本(控制台粘贴运行)
- `icons/icon-128.png` - 工具栏图标
